# AquaGrid Core - Requirements

## Backend
- Java 21
- Spring Boot 4.0+
- Spring Web
- Spring Data JPA
- Spring Validation

## Database
- PostgreSQL 16
- JDBC Driver (org.postgresql)

## ORM
- Hibernate (bundled with Spring Data JPA)

## Build Tool
- Maven

## IDE
- IntelliJ IDEA

## Optional (Future)
- PostGIS (for GIS features)
- Lombok (reduce boilerplate)
- Spring Boot Actuator (monitoring)
